> 为了规范代码生成，所以在特定的字段，用特定的方式来生成

### 创建外键
```
SchemaHelper::foreignId($table, 'company_id', 'companies');  
```
#### 创建枚举
```
SchemaHelper::Enum($table, 'status', CompanyAdminsStatusEnum::class, '状态', default: CompanyAdminsStatusEnum::Enabled->value, nullable: false);
```
#### 其他自己研究源码

#### 例子

```
Schema::create('company_admins', function (Blueprint $table) {  
    $table->id();  
    SchemaHelper::foreignId($table, 'company_id', 'companies');  
  
    $table->string('username', 20)->comment('用户名');  
    $table->string('password', 100)->comment('密码');  
  
    SchemaHelper::Enum($table, 'status', CompanyAdminsStatusEnum::class, '状态', default: CompanyAdminsStatusEnum::Enabled->value, nullable: false);  
    $table->timestamps();  
});
```
